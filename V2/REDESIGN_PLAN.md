# ПЛАН ПЕРЕДЕЛКИ ДИЗАЙНА
## Горизонтальный скролл timeline с единой линией

---

## Проблемы текущего дизайна

1. ❌ Все элементы накладываются друг на друга
2. ❌ Нет горизонтального скролла
3. ❌ Слишком много компонентов
4. ❌ Используются градиенты (не соответствует требованиям)
5. ❌ Не видно полный жизненный цикл этапов оплаты

---

## Новый дизайн

### Концепция
**Горизонтальный скролл с единой длинной линией**, где:
- Месяцы двигаются слева направо при скролле
- Видно весь жизненный цикл этапов оплаты
- Минималистичный современный дизайн без градиентов
- Меньше компонентов, больше чистоты

### Структура этапов

1. **Месяцы 1-3**: Этапы разработки (100 000 AMD)
   - Цвет: Синий (#3B82F6)
   - Подпись: "Этап разработки"

2. **Gate**: Передача проекта
   - Вертикальная линия с иконкой
   - Подпись: "Проект сдан"

3. **Месяцы 4-12**: Subscription + Maintenance (150 000 AMD)
   - Цвет: Зеленый (#10B981)
   - Подпись: "Subscription + Maintenance"
   - Показывать все месяцы: 4, 5, 6, 7, 8, 9, 10, 11, 12

4. **После 12 месяцев**: Только Maintenance (50 000 AMD)
   - Цвет: Фиолетовый (#8B5CF6)
   - Подпись: "Только Maintenance"

## Техническая реализация

### Вариант 1: CSS Grid + Horizontal Scroll (РЕКОМЕНДУЕТСЯ)
**Плюсы:**
- Чистый CSS, без зависимостей
- Отличная производительность
- Легко поддерживать
- Нативный горизонтальный скролл

**Минусы:**
- Нужно правильно настроить ширину контейнера

**Профессиональность:** ⭐⭐⭐⭐⭐

### Вариант 2: Flexbox + Horizontal Scroll
**Плюсы:**
- Простая реализация
- Хорошая поддержка браузерами

**Минусы:**
- Менее гибкий чем Grid

**Профессиональность:** ⭐⭐⭐⭐

## Этапы реализации

### Этап 1: Структура HTML
- Создать контейнер с горизонтальным скроллом
- Разместить все месяцы (1-12) + Gate + Maintenance
- Упростить структуру компонентов

### Этап 2: CSS - Единая линия
- Создать длинную горизонтальную линию
- Разместить точки этапов на линии
- Настроить горизонтальный скролл

### Этап 3: CSS - Карточки этапов
- Упростить карточки (без градиентов)
- Использовать чистые цвета
- Минималистичный дизайн

### Этап 4: JavaScript - Интерактивность
- Анимация при скролле
- Активация этапов после Gate
- Плавные переходы

### Этап 5: Полировка
- Оптимизация производительности
- Адаптивность для мобильных
- Финальные правки

## Дизайн-система

### Цвета (чистые, без градиентов)
- **Разработка**: #3B82F6 (синий)
- **Subscription + Maintenance**: #10B981 (зеленый)
- **Только Maintenance**: #8B5CF6 (фиолетовый)
- **Заблокировано**: #9CA3AF (серый)
- **Фон**: #FFFFFF (белый)
- **Текст**: #111827 (темно-серый)

### Типографика
- Шрифт: Inter
- Заголовки: 700
- Текст: 500
- Мелкий текст: 400

### Компоненты
- Карточки: простые, с рамкой, без теней
- Точки: круглые, чистые цвета
- Линия: простая, 2px толщина
